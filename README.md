### 🚀 ব্যবহৃত প্রযুক্তি (Technologies Used)

প্রকল্পটি অত্যন্ত ক্লিন কোড এবং মডার্ণ আর্কিটেকচার অনুসরণ করে তৈরি করা হয়েছে:

* **Frontend Core:** React.js (Vite) ব্যবহার করে হাই-পারফরম্যান্স নিশ্চিত করা হয়েছে।
* **Styling:** Tailwind CSS ব্যবহার করে সম্পূর্ণ রেসপন্সিভ এবং মডার্ন UI ডিজাইন করা হয়েছে।
* **Navigation:** React Router Dom এর মাধ্যমে সিঙ্গেল পেজ অ্যাপ্লিকেশন (SPA) ফ্লো বজায় রাখা হয়েছে।
* **Data Visualization:** Recharts লাইব্রেরি ব্যবহার করে ইন্টারঅ্যাকশন ডাটা গ্রাফিক্যাল রিপ্রেজেন্টেশন করা হয়েছে।
* **State & Storage:** LocalStorage ব্যবহার করে ডাটা পারসিস্টেন্স বা স্থায়ীত্ব নিশ্চিত করা হয়েছে।

### ✨ প্রজেক্টের মূল বৈশিষ্ট্যসমূহ (Core Features)

#### 1. 📅 Dynamic Timeline Logging
প্রতিটি ফ্রেন্ড প্রোফাইলে গিয়ে আপনি কল, টেক্সট বা ভিডিও কলের রেকর্ড রাখতে পারেন। এটি অটোমেটিক গ্লোবাল টাইমলাইনে তারিখ অনুযায়ী জমা হয়, যা আপনার যোগাযোগের ইতিহাস সংরক্ষণ করে।

#### 2. 📊 Friendship Analytics (Visual Data)
Stats পেজে একটি প্রিমিয়াম পাই-চার্ট (Pie Chart) যুক্ত করা হয়েছে। এটি আপনার মোট যোগাযোগের ধরন (Call, Text, Video) বিশ্লেষণ করে রিয়েল-টাইম ডাটা প্রদর্শন করে।

#### 3. 🔴 Smart Status Tracking
বন্ধুদের সাথে যোগাযোগের প্রয়োজনীয়তা বোঝাতে ৩টি ডাইনামিক স্ট্যাটাস ব্যবহার করা হয়েছে: 
- **On-track:** নিয়মিত যোগাযোগ আছে।
- **Almost due:** যোগাযোগ করার সময় হয়েছে।
- **Overdue:** দ্রুত যোগাযোগ করা প্রয়োজন।
